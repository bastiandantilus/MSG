**MSG!** 
--------
*by Scott Lawrence (theelven@gmail.com)*

A minimalist match 3 game for Ludum Dare 26

Swap the blocks to make matches based on number, color, or symbol. Match Set Game!

Features
--------

 - Swap blocks!
 - Make Matches!
 - Score points!
